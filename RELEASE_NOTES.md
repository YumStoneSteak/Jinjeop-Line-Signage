# Jinjeop Line Signage v2.3.0

## 주요 변경

- NSIS 설치 파일명을 `Jinjeop.Line.Signage.v2.3.0.exe` 형식으로 정리했습니다.
- 설치 앱 실행 파일과 제거 표시명은 영문 `Jinjeop Line Signage` 기준으로 통일했습니다.
- 바탕화면 바로가기는 `진접선 행선안내 사이니지 v2.3.0` 이름과 내장 `ncuc.ico` 아이콘을 사용합니다.
- 2.0.0/2.0.1/2.2.0 실행 중 업그레이드가 막히지 않도록 설치 시작 시 기존 앱 프로세스를 종료합니다.
- 첫 설치 또는 초기화 후 현재 역을 필수로 선택하고 저장해야 다른 기능을 사용할 수 있도록 했습니다.
- 패널 좌우 전환 기능을 제거했습니다.
- 설정 도움말 물음표를 줄이고, 툴팁이 설정 화면 밖에서 잘리지 않도록 수정했습니다.
- 진접선 열차 방향 범례, 상선/하선 아이콘 파일명, 노선 라인 스타일을 정리했습니다.
- 열차 위치 정보 자동 흐름과 진단 로그를 보강했습니다.
- `run-dev-check.bat` 개발 실행 스크립트를 `npm.cmd` 기반으로 수정했습니다.
- 이후 배포를 직접 수행할 수 있도록 `release-github.bat`와 배포 안내 문서를 추가했습니다.

## 배포 파일

- `Jinjeop.Line.Signage.v2.3.0.exe`
- `Jinjeop.Line.Signage.v2.3.0.exe.blockmap`
- `latest.yml`
