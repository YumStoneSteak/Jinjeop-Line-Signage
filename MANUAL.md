# 진접선 행선안내 사이니지 사용자 매뉴얼

처음 사용하는 운영자가 상단 메뉴 순서대로 필요한 설정을 확인할 수 있도록 정리한 간단 매뉴얼입니다.
설정 화면의 각 항목 옆 `?` 아이콘에 마우스를 올리면 이 매뉴얼과 같은 기준의 짧은 도움말을 바로 확인할 수 있습니다.

## 1. 실행

1. GitHub Release에서 받은 `Jinjeop Line Signage v2.2.1.exe`를 설치합니다.
2. 시작 메뉴 또는 바탕화면 바로가기로 앱을 실행합니다.
3. 운영 화면으로 전환하려면 상단 메뉴의 **사이니지 모드 시작**을 누릅니다.
4. 설정 메뉴를 다시 보려면 화면 맨 위로 마우스를 이동합니다.
5. 운영 화면에서 빠져나오려면 **사이니지 모드 종료**를 누릅니다.

## 2. 일반 설정

상단 메뉴에서 **일반 설정**을 엽니다.

### 레이아웃

- **분할 비율**: 열차 위치 정보와 공지 영역의 가로 비율을 선택합니다.

### 창 동작

- **항상 최상위 유지**: 다른 창보다 앱을 앞에 둡니다.
- **최소화 방지**: 실수로 앱이 최소화되는 것을 막습니다.
- **앱 시작 시 전체 화면 모드**: 실행 직후 운영 화면으로 시작합니다.
- **Windows 시작 시 자동 실행**: PC 부팅 후 앱을 자동 실행합니다.

### 운영 시간 / 자동 업데이트

- **자동 업데이트**: 켜져 있으면 지정된 시간에 새 버전을 확인하고 설치합니다.
- **업데이트 시간**: 기본 `01:00`입니다. PC 종료 전에 업데이트하려면 PC 종료 시간보다 앞선 시간으로 둡니다.
- **PC 종료 시간**: 기본 `02:00`입니다. 이 시간부터 자동 작업을 보류합니다.
- **PC 부팅 시간**: 기본 `04:30`입니다. 이 시간 이후 자동 작업을 다시 시작합니다.
- **업데이트 확인**: 새 버전이 있는지 즉시 확인합니다.
- **지금 설치**: 다운로드된 업데이트가 있으면 바로 설치합니다. 없으면 확인 후 설치를 준비합니다.

앱은 PC를 종료하거나 부팅하지 않습니다. 이 시간은 앱이 자동 작업을 해도 되는지 판단하는 기준입니다.

### 관리

- **관리자 옵션**: 로고 표시, 현재 역명 표시, 로고 파일 경로, 좌측 바 너비 같은 고급 항목을 표시합니다.

변경 후 **저장**을 누릅니다.

## 3. 위젯

상단 메뉴에서 **위젯**을 엽니다.

- 표시할 좌측 정보 위젯을 선택합니다.
- 위젯 순서를 드래그로 바꿉니다.
- 현재 역은 필수 항목입니다. 첫 설치 또는 초기화 후 현재 역을 선택하고 저장해야 다른 기능을 사용할 수 있습니다.
- 표시 방향, 시간표 표시 방식을 설정합니다.
- 날씨 위젯은 계절에 따라 최고/최저 또는 최저/최고 순서를 자동으로 바꿉니다.

변경 후 **저장**을 누릅니다.

## 4. 열차 위치 정보

상단 메뉴에서 **열차 위치 정보**를 엽니다.

- **시작 URL**: 열차 위치 정보 화면 주소입니다. 특별한 이유가 없으면 기본값을 유지합니다.
- **기본 웹 확대/축소(%)**: 기본값은 `125`입니다.
- **자동 새로고침 사용**: 지정한 주기마다 열차 위치 정보 화면을 새로고침하고 4호선, 확대, 드래그 보정을 다시 적용합니다.
- **자동 새로고침 주기**: 기본값은 6시간입니다. PC 종료 시간대에는 실행하지 않습니다.
- **팝업 처리**: 운영 중 불필요한 팝업을 막거나 현재 화면에서 열도록 설정합니다.
- **드래그 보정**: 새로고침 후 화면 위치를 자동 보정합니다.

운영 불가능 시간대에는 자동 새로고침이 보류됩니다.

## 5. 공지

상단 메뉴에서 **공지**를 엽니다.

- **파일 추가**: 이미지 또는 영상을 공지 목록에 추가합니다.
- **누락 파일 확인**: 삭제되거나 이동된 파일을 확인합니다.
- **재생시간**: 각 이미지 또는 영상의 표시 시간을 설정합니다.
- **게시 시작일 / 게시 종료일**: 특정 기간에만 공지를 표시합니다.
- 목록을 드래그해 재생 순서를 바꿉니다.

## 6. 사이니지 모드

- **사이니지 모드 시작**: 설정 메뉴를 숨기고 운영 화면으로 전환합니다.
- **사이니지 모드 종료**: 설정 가능한 일반 화면으로 돌아옵니다.
- 상단 우측 창 제어 버튼으로 최소화, 창 크기 전환, 종료를 할 수 있습니다.

## 7. 문제 해결

- 업데이트가 되지 않으면 **일반 설정 > 운영 시간 / 자동 업데이트**에서 업데이트 시간이 자동 작업 보류 시간대 안에 들어가 있지 않은지 확인합니다.
- 자동 실행이 되지 않으면 **일반 설정 > Windows 시작프로그램 설정 창 열기**에서 앱 허용 상태를 확인합니다.
- 열차 위치 정보가 이상하면 **열차 위치 정보** 메뉴에서 저장 후 새로고침하거나 앱을 다시 실행합니다.
- 공지가 보이지 않으면 **공지 > 누락 파일 확인**을 실행합니다.
