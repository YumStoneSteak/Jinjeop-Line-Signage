# Jinjeop Line Signage 사용자 매뉴얼

이 문서는 처음 사용하는 운영자가 상단 메뉴 순서대로 필요한 설정을 빠르게 찾을 수 있도록 만든 간단 매뉴얼입니다.

## 1. 기본 실행

1. GitHub 릴리즈에서 받은 `Jinjeop Line Signage Setup 2.0.2.exe`를 실행해 설치합니다.
2. 설치 후 시작 메뉴 또는 바탕화면 바로가기로 앱을 실행합니다.
3. 앱이 열리면 상단 메뉴에서 필요한 설정을 조정합니다.
4. 운영 화면으로 전환하려면 **사이니지 모드 시작**을 누릅니다.
5. 사이니지 운영 중 상단 메뉴를 다시 보려면 화면 맨 위쪽으로 마우스를 이동합니다.
6. 운영 화면을 끝내려면 **사이니지 모드 종료**를 누릅니다.

개발 또는 소스 폴더에서 직접 실행할 때는 `run-dashboard.bat`를 사용할 수 있습니다.

## 2. 일반 설정

상단 메뉴의 **일반 설정**을 누릅니다.

### 레이아웃

- **분할 비율**: 열차 위치 정보 영역과 공지 게시판 영역의 가로 비율을 선택합니다.
- **패널 좌우 전환**: 열차 위치 정보와 공지 게시판의 좌우 위치를 바꿉니다.

### 창 동작

- **항상 최상단 유지**: 다른 창보다 사이니지 앱을 위에 유지합니다.
- **최소화 방지**: 실수로 창이 최소화되는 것을 막습니다.
- **앱 시작 시 전체 화면 모드**: 앱 실행 직후 사이니지 화면으로 시작합니다.
- **Windows 시작 시 자동 실행**: PC 부팅 후 앱을 자동 실행합니다.
- **시작프로그램 폴더 열기**: 자동 실행 바로가기 위치를 확인합니다.
- **윈도우 시작프로그램 설정 창 열기**: Windows 시작 앱 허용 상태를 확인합니다.

### 관리

- **관리자 옵션**: 로고 표시, 현재 역명 표시, 로고 파일 경로, 좌측 바 너비 같은 고급 항목을 보여줍니다.
- 일반 사용자는 관리자 옵션을 꺼둔 상태로 사용하는 것을 권장합니다.

변경 후 **저장**을 누릅니다.

## 3. 위젯

상단 메뉴의 **위젯**을 누릅니다.

### 위젯 순서

- 왼쪽 핸들을 드래그해 좌측 정보 영역의 표시 순서를 바꿉니다.
- 운영 화면에서 필요한 정보가 위쪽에 오도록 배치합니다.

### 기본 정보 설정

- **현재 역**: 별내별가람역, 오남역, 진접역 중 표시할 역을 선택합니다.
- 관리자 옵션을 켜면 로고 표시, 현재 역명 표시, 로고 파일 경로, 좌측 바 너비를 조정할 수 있습니다.

### 날씨 위젯

- 날씨 정보는 자동 갱신됩니다.
- 최고/최저 기온 순서는 계절 기준으로 자동 변경됩니다.

### 멀티 위젯 설정

- **멀티 위젯 표시**: 24절기와 오늘의 한마디를 한 칸에서 번갈아 표시합니다.
- **전환 효과**와 **순환 시간**을 설정할 수 있습니다.

### 다음 열차 설정

- **표시 방향**: 상행 또는 하행을 선택합니다.
- **표시 형식**: 표형식 또는 시간형식을 선택합니다.
- **시간표 갱신**: 로컬 시간표 캐시를 새로 갱신합니다.

변경 후 **저장**을 누릅니다.

## 4. 열차 위치 정보

상단 메뉴의 **열차 위치 정보**를 누릅니다.

### 기본 설정

- **시작 URL**: 열차 위치 정보 웹 화면 주소입니다. 특별한 이유가 없으면 기본값을 유지합니다.
- **기본 웹 확대/축소(%)**: 기본값은 `125`입니다.
- **자동 새로고침 사용**: 일정 시간마다 열차 위치 정보 화면을 새로고침합니다.
- **자동 새로고침 주기(시간)**: 기본값은 6시간입니다.

### 팝업

- **웹사이트 팝업 금지**: 운영 중 불필요한 팝업을 막습니다.
- 팝업을 허용할 경우 별도 창 또는 현재 뷰에서 여는 방식을 선택할 수 있습니다.

### 드래그 보정

- **새로고침 후 드래그 보정 사용**: 새로고침 후 지도 위치를 자동으로 보정합니다.
- **드래그 녹화 시작**: 실제 화면에서 원하는 드래그를 녹화합니다.
- **녹화 삭제**: 저장된 드래그 보정값을 지웁니다.
- **기본값으로 저장**: 현재 보정값을 기본값으로 저장합니다.
- 수동 입력 항목은 시작 X/Y, 끝 X/Y, 시간(ms) 순서입니다.

변경 후 **저장**을 누릅니다.

## 5. 공지

상단 메뉴의 **공지**를 누릅니다.

### 파일 작업

- **파일 추가**: 공지에 사용할 이미지 또는 영상을 추가합니다.
- **누락 파일 확인**: 이동되거나 삭제된 파일이 있는지 확인합니다.

### 공지 재생

- **전환 효과**: 공지 전환 방식을 선택합니다.

### 재생 목록

- **재생시간(초)**: 각 이미지 또는 영상의 표시 시간을 설정합니다.
- **게시 시작일 / 게시 종료일**: 특정 기간에만 공지를 보이게 설정합니다.
- **상태**: 현재 게시 가능한지 확인합니다.
- **삭제**: 목록에서 해당 공지를 제거합니다.
- 행의 왼쪽 핸들을 드래그하면 공지 재생 순서를 바꿀 수 있습니다.

변경 후 패널을 닫으면 설정이 반영됩니다.

## 6. 사이니지 모드 시작/종료

- **사이니지 모드 시작**: 운영 화면으로 전환합니다.
- **사이니지 모드 종료**: 설정 가능한 일반 화면으로 돌아옵니다.
- 버튼 왼쪽과 오른쪽의 구분선은 설정 메뉴, 사이니지 모드, 창 제어 영역을 구분합니다.

## 7. 창 제어 버튼

상단 오른쪽의 아이콘 버튼을 사용합니다.

- `-`: 창 최소화
- `전체 화면 아이콘`: 창 크기 전환
- `X`: 앱 종료

## 8. 문제가 생겼을 때

- 열차 위치 정보가 이상하면 **열차 위치 정보** 메뉴에서 저장 후 새로고침하거나 앱을 다시 실행합니다.
- 공지가 보이지 않으면 **공지 > 누락 파일 확인**을 실행합니다.
- 자동 실행이 되지 않으면 **일반 설정 > 윈도우 시작프로그램 설정 창 열기**에서 앱 허용 상태를 확인합니다.
- 화면 배치가 이상하면 **일반 설정 초기화**, **위젯 설정 초기화**, **열차 위치 정보 설정 초기화**를 순서대로 확인합니다.
