# Jinjeop Line Signage

진접선 행선안내 사이니지는 남양주도시공사 안내 PC에서 사용할 수 있도록 만든 Windows용 Electron 사이니지 앱입니다.

왼쪽에는 로고, 현재 역명, 날짜/시간, 날씨, 24절기, 오늘의 한마디, 열차 시간표 위젯을 표시하고, 가운데에는 진접선 열차 위치 정보를, 오른쪽에는 공지 이미지 또는 영상을 표시합니다.

## 현재 버전

v2.0.1

## v2.0.1 주요 내용

- Windows 시작 시 자동 실행과 앱 시작 시 전체 화면 모드를 기본값으로 사용합니다.
- 자동 실행 상태 확인, 시작프로그램 폴더 열기, Windows 시작프로그램 설정 열기 기능을 제공합니다.
- 기본 웹 확대율을 125%로 통일했습니다. 내부 값이 `1.25` 또는 `125`로 저장되어 있어도 정상 인식합니다.
- 열차 위치 정보 웹뷰 자동 새로고침과 4호선 화면 자동 진입 흐름을 강화했습니다.
- 새로고침 후 지도 위치를 보정하는 드래그 보정값을 설정 화면에서 확인하고 저장할 수 있습니다.
- 날씨 위젯의 최고/최저 기온 표시 순서를 월 기준으로 자동 변경합니다.
  - 4월부터 10월까지: 최고 / 최저
  - 11월부터 3월까지: 최저 / 최고
- 시간 표시는 12시간제 기준으로 표시하며, 한 자리 시간도 두 자리 폭을 유지해 화면이 흔들리지 않습니다.
- 관리자 옵션이 꺼져 있을 때 로고 표시, 현재 역명 표시 등 관리자 전용 항목이 숨겨집니다.
- 좌측 로고 위치를 오른쪽 제목 영역과 같은 높이로 맞추고, 하단 위젯 위치는 유지했습니다.
- 사이니지 모드 버튼 좌우에 구분선을 배치해 설정 메뉴와 창 제어 영역을 구분했습니다.
- 좌측 위젯 순서, 위젯 표시 여부, 멀티 위젯, 공지 게시 기간 등 운영 설정을 저장할 수 있습니다.

## 화면 구성

- **좌측 정보 영역**: 로고, 현재 역명, 날짜/시간, 날씨, 멀티 위젯, 열차 시간표 등
- **진접선 열차 위치 정보**: 서울교통공사 열차 위치 정보 웹 화면
- **공지 게시판**: 이미지와 영상을 순서대로 재생하는 공지 영역
- **상단 메뉴**: 일반 설정, 위젯, 열차 위치 정보, 공지, 사이니지 모드 시작/종료

## 빠른 실행

### 1. 의존성 설치

```powershell
npm install
```

### 2. 개발 실행

```powershell
npm start
```

### 3. 운영 실행

```powershell
.\run-dashboard.bat
```

`run-dashboard.bat`는 필요한 의존성이 없으면 설치 안내를 표시하고, 준비가 끝나면 앱을 실행합니다.

## Windows 빌드

전체 Windows 빌드는 다음 명령으로 실행합니다.

```powershell
npm run build
```

빌드 결과는 `dist` 폴더에 생성됩니다.

portable 실행 파일만 빠르게 만들 때는 다음 파일을 실행합니다.

```powershell
.\build-portable.bat
```

v2.0.1 릴리즈 빌드 파일 이름은 보통 다음 형식입니다.

```text
dist\Jinjeop Line Signage 2.0.1.exe
```

## 주요 메뉴

자세한 사용 방법은 [MANUAL.md](MANUAL.md)를 확인하세요.

- **일반 설정**: 화면 분할, 창 동작, 자동 실행, 관리자 옵션
- **위젯**: 좌측 위젯 순서, 기본 정보, 날씨, 멀티 위젯, 열차 시간표
- **열차 위치 정보**: 시작 URL, 확대율, 자동 새로고침, 팝업 차단, 드래그 보정
- **공지**: 공지 파일 추가, 재생 효과, 재생 시간, 게시 기간, 순서 변경
- **사이니지 모드 시작/종료**: 운영 화면 전체화면 전환

## 파일 위치

- 기본 로고: `files/logos/ncuc_logo.png`
- 공지 이미지/영상: `files/notices/`
- 일반 이미지 리소스: `files/images/`
- 빌드 아이콘: `files/icons/ncuc.ico`
- 설정 파일: Electron `userData` 경로의 `config.json`
- 시간표/절기 캐시: Electron `userData` 경로 내부 캐시 파일

## 운영 참고

- 설정 변경 후에는 각 패널의 **저장** 버튼을 눌러야 다음 실행 때 복원됩니다.
- Windows 시작 시 자동 실행은 기본으로 켜져 있습니다. Windows 보안 또는 시작 앱 정책에 따라 첫 실행 후 사용자가 허용해야 할 수 있습니다.
- 열차 위치 정보는 외부 웹사이트 상태와 네트워크 환경의 영향을 받습니다.
- 공지 파일 경로가 바뀌면 공지 패널에서 누락 파일 확인을 실행하세요.
- Git에는 운영 PC의 개인 설정 파일, 로그, 캐시 파일을 포함하지 않습니다.
